typescript
==========

Typescript 2.0之后，tsd和typings都可以去掉了。要获得lodash的类型定义文件只需要
```text
npm install @types/lodash
```
这样一来，typescript的工作流就和普通的Node.js项目没什么区别了。  
更重要的是Typescript 2.1之后，async/await可以直接编译到ES5。babel什么的，再见吧
